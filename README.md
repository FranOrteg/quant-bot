## 🚀 Instalación y Configuración

### Requisitos
```bash
pip install -r requirements.txt
```

### Variables de Entorno
Crear archivo `.env` en la raíz del proyecto:
```env
BINANCE_API_KEY=tu_api_key_aqui
BINANCE_API_SECRET=tu_api_secret_aqui
```
